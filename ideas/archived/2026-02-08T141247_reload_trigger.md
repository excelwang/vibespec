# Idea: Reload Trigger

**Timestamp**: 2026-02-08T14:12:47+08:00

## Proposal

When user inputs `vibespec reload`, vibe-spec MUST reload its own definition file (SKILL.md).

## Rationale

- Allows hot-reload of skill changes without restarting session
- Essential for self-bootstrapping workflow

## Target

- L1: Add `CONTRACTS.RELOAD_TRIGGER`
- SKILL.md: Add `vibespec reload` trigger section
