# Idea: Direct Bug RCA Mode

**Timestamp**: 2026-02-08T16:42:49+08:00

## Proposal

When user inputs `vibespec bug [description]`, the system MUST **Skip Idea Capture** and immediately enter **RCA Mode**.

**Workflow**:
1. User: `vibespec bug tests fail on CI`
2. Agent: "Starting RCA on 'tests fail on CI'..."
3. Agent: Scans specs, logs, code. (RCA Steps)
4. Agent: "Root Cause Found: Spec Gap in L2."
5. Agent: "Proposed Fix Idea: `fix_l2_ci_gap.md` Created."

## Rationale

- **Efficiency**: "Bug" implies urgency. Capturing an idea first is bureaucratic overhead.
- **Context**: The bug description IS the context for analysis.

## Target

- **SKILL**: Update `vibespec bug` trigger behavior.
- **L1**: Update `TRIGGER_BUG` contract (Direct Action).
